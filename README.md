# Estrutura de Dados - Implementação de Fila (Queue)

Este repositório contém a implementação de uma estrutura de dados do tipo **Fila (FIFO)** em Python. O próprio arquivo principal demonstra o uso prático da classe e o tratamento correto de erros através de estruturas `try/except`.

## Como Executar

### Pré-requisitos
Certifique-se de ter o Python 3.x instalado no seu computador.

### Executando o Arquivo
Abra o terminal na pasta do arquivo e execute o comando abaixo. Ele rodará a classe e exibirá na tela os fluxos de sucesso e as capturas de erros simuladas:

```bash
python queue.py
